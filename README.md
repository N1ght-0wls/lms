# Awesome LMS

> [!WARNING]  
> The app is a work in progress still. We're building it in public

## Development

Install dependencies

```sh
pnpm i
```

Build all app's packages

```sh
pnpm app:prepare
```

Start development server

```sh
pnpm dev
```
